This program uses the WAVEngineLib library to perform the following audio file conversions:

  1. List: This option prints the details of the given music file(s)
  2. Stereo To Mono: This option changes music file(s) from stereo to mono
  3. Mix: This option mixes the two wave files into one
  4. Chop: This option cuts the given music file and from the start second to the end second
  5. Reverse: This option reverses the given music files
  6. Encode Text: This option encodes the given text (inside the input file) to the given wave file
  7. Decode Text: This option decodes the given music file and prints the hidden message with the given length to the output file
  8. Change Speed: This option multiplies the speed of the given music file(s) with the given number
  9. Volume Increase: This option multiplies the volume of the given music file(s) by the number given

COMPILE COMMAND:

  • To create the library use the command: "make team9"
  • After this, compile the file using the library (gcc -o wavengine wavengine.c epl232_team9.a)

EXECUTION COMMANDS:

  • ./wavengine -list <wave file(s)>
  • ./wavengine -mono <wave file(s)>
  • ./wavengine -mix <wave file 1> <wave file 2>
  • ./wavengine -chop <wave file> <start second> <end second>
  • ./wavengine -reverse <wave file(s)>
  • ./wavengine -encodeText <wave file> <input file with text to encode>
  • ./wavengine -decodeText <wave file> <message length> <output file>
  • ./wavenging -changeSpeed <number to be multiplied> <wave file(s)>
  • ./wavengine -volIncrease <number to be multiplied> <wave file(s)>
  • ./wavengine -man

NOTE: All options do not change the music files, they create new music files (except -list, -decode)
