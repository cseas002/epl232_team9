int mix ( char const *  filename1,
char const *  filename2 
)

This method merges two audio files into one.

It reads the header and data from both files. It creates a new audio file using the left channel of the first file and the right channel of the second one. The size of the new audio file is the size of the smaller file. If the two files do not have the same number of bits per sample or either of those is mono, then the function terminates.

Parameters
filename1The name of the first file (left channel).
filename2The name of the second file (right channel).
Returns
An exit code.
